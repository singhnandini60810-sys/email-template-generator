# Email Template Generator

A simple web application that generates professional email templates for internship requests, job applications, and leave applications.

## Features

- Internship email generator
- Job application email generator
- Leave application generator
- Copy to clipboard
- Clean and simple UI
- Works offline

## Tech Stack

- HTML
- CSS
- JavaScript

## How to Use

1. Select email type
2. Enter name and details
3. Click generate
4. Copy the email

## Future Improvements

- Add more email templates
- Download email option
- Dark/light theme
- Custom subject editing
